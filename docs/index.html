FAQ Project

A 5001 CEM Project focused on creating a dynamic website to allows user to post questions with images and answers.

To start the server, run the command node index.js from the root directory and navigate to localhost:8080 in the browser to display the website.

The repository of the project is hosted at: https://github.coventry.ac.uk/5001CEM-1920SEPJAN/Frequently-Asked-Questions
